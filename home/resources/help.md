# XMonad Key Bindings

Caps Lock +
- q         : kill window
- w         : web browser
- e         : emacs
- r         : reload xmonad
- a         : terminal
- s         : scrachpad
- d         : dmenu
- z         : change layout
- /         : help
- j         : cycle window down
- k         : cycle window up
- Shift + q : quit exmonad
- Space     : switch between α and β workspaces
- m         : switch to the μ workspace
- u,i,o,p   : switch to the α,β,τ,μ workspace
